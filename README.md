## npm install 

安装依赖包

```
superagent(http://visionmedia.github.io/superagent/ ) http 库，可以发起 get 或 post 请求。
cheerio(https://github.com/cheeriojs/cheerio )  Node.js 版的 jquery。
```

最后运行：node app.js
